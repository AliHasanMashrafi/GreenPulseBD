## 🌍 What is GreenPulse BD?

**GreenPulse BD** is a climate intelligence web platform that uses **real NASA satellite data** to visualize and analyze the **Urban Heat Island (UHI) effect** across Dhaka city, Bangladesh.

The platform answers three key questions for every one of Dhaka's 46 administrative *thanas* (zones):
1. **How hot is it?** — Land Surface Temperature (LST) from NASA Landsat
2. **How green is it?** — Vegetation density (NDVI) from satellite imagery
3. **What can be done?** — AI-generated, budget-aware greening recommendations

---

## 🚨 The Problem

Dhaka is one of the **most heat-stressed megacities in South Asia**:

- 🌡️ Urban surface temperatures regularly exceed **40°C** in summer
- 🌿 Most thanas fall **far below** the WHO-recommended **25% green coverage** threshold
- 🏘️ Dense, low-income neighborhoods have virtually **no green space**
- 📉 Urban green cover has been **declining** over the past 5 years
- ❌ No accessible, data-driven tool existed for citizens to understand **their** local climate risk

## 🛰️ Data Source & Scientific Methodology

| Parameter | Details |
|---|---|
| **Satellites** | NASA Landsat 8 & Landsat 9 (Collection 2, Tier 1, Level 2) |
| **Processing Platform** | Google Earth Engine (GEE) |
| **Key Metrics** | LST (Land Surface Temperature in °C) · NDVI (Normalized Difference Vegetation Index) |
| **Data Period** | March – April 2026 (multi-scene composite average) |
| **Processed On** | May 2, 2026 |
| **Coverage** | All 46 administrative thanas of Dhaka City Corporation |
| **Data Format** | Curated JSON (`src/data/dhaka_heat_data.json`) |
| **Historical Data** | 5-year LST trend (2021–2025) per thana from archived Landsat imagery |

> ⚠️ **Scientific Note:** The displayed temperature and vegetation data represent a **March–April 2026 composite average** — not a single snapshot. This multi-scene composite approach reduces cloud interference and provides statistically robust values for each thana. Values are area-averaged means, not point measurements.

## ✨ Features in Detail

### 🌡️ Thermal Heatmap Page
- Interactive Leaflet.js map with all 46 Dhaka thanas color-coded by LST
- Color scale: Yellow → Orange → Deep Red → Crimson (by temperature severity)
- Click any thana to see: LST, heat level, 5-year trend bar chart, weather advisory
- Per-area heat rank (1 = hottest in Dhaka)
- Data methodology panel showing satellite source and data period

### 🌿 Green Space Analyzer Page
- Same map in "green mode" — Yellow to Deep Green color scale (by NDVI %)
- Click any thana: greenery percentage, 5-year trend, greenery rank
- **Rooftop Space card** — estimated % of rooftop area available for gardening
- **Open Space card** — greening potential score for roadside/open plots
- Personalized recommendations per thana (e.g., "Rooftop Kitchen Garden", "Native Trees")

### 🤖 AI Green Planner (Gemini-powered)
- Form inputs: Area → Space Type (Rooftop / Balcony / Open Field) → Priority → Budget → Size
- **Priority**: পরিবেশ (Environment) · ভারসাম্য (Balanced) · অর্থনৈতিক (Economic)
- GPS auto-detection to pre-fill nearest thana
- Generates 3 plan tiers: **Low Cost | Standard | Premium**
- Each plan: Bengali title, description, CO₂ offset, O₂ production, monthly profit, cooling effect
- Suggested plants (Bengali names) + implementation tools
- Fallback to secondary API key if primary quota is exceeded

### 🏠 Homepage
- Dynamically ranks **Top 5 Hottest Areas** from real satellite data
- Live map preview (non-interactive, performance-optimized)
- Data provenance: last processed date shown
- CTA sections linking to all platform features

## ⚠️ Known Limitations (Prototype)

This is a **hackathon prototype** built in under 2.5 hours. The following limitations exist:

| Limitation | Description |
|---|---|
| 📅 **Static Data** | Data is a manually curated JSON; not live/auto-refreshing from NASA APIs |
| 🗓️ **Data Period** | Represents March–April 2026 composite, not real-time or daily updates |
| 🌡️ **LST Approximation** | Area-averaged LST values; exact street-level temperatures may vary |
| 🌿 **NDVI Accuracy** | NDVI is computed from optical bands; cloud cover during composite period may affect accuracy |
| 📍 **Boundary Precision** | Thana boundaries are approximated polygons, not official DCC cadastral data |
| 🤖 **AI Plans** | Gemini-generated plans are advisory estimates; not professionally certified |
| 💰 **Cost Estimates** | Budget figures in AI plans are rough estimates based on Dhaka market rates (2026) |
| 📱 **Mobile Optimization** | Partially optimized; map interaction may be limited on small screens |
| 🔄 **"প্রজেক্ট শুরু করুন"** | Project initiation button is a prototype placeholder — full feature planned |
| 🏠 **Roof Planning** | The roof-planning page is in early prototype stage |
| 🔑 **API Dependency** | AI Planner requires a valid Gemini API key; features degrade without it |

## ⚙️ Getting Started

### Prerequisites
- **Node.js** 18 or higher
- **npm** 9 or higher
- A **Google Gemini API key** (free tier available at [Google AI Studio](https://aistudio.google.com))

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/AliHasanMashrafi/GreenPulseBD.git
cd GreenPulseBD

# 2. Install dependencies
npm install

# 3. Set up environment variables
cp .env.example .env.local
# Then edit .env.local and add your Gemini API key
```

### Environment Variables

```bash
# .env.local
NEXT_PUBLIC_GEMINI_API_KEY=your_gemini_api_key_here
NEXT_PUBLIC_GEMINI_API_KEY_2=your_backup_api_key_here  # Optional fallback
```

Get a free API key at: https://aistudio.google.com/app/apikey

```bash
# 4. Start the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

### Production Build

```bash
npm run build
npm run start
```

## 📊 Data Update Guide

To update the satellite dataset:

1. Run your GEE (Google Earth Engine) script to export new LST/NDVI data
2. Process the output into the required JSON schema (see below)
3. Replace `src/data/dhaka_heat_data.json` with the new file
4. Restart the Next.js server — no code changes needed
